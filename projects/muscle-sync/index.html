Muscle SYNC

AI-based sensorless muscle activation analysis

Website

Overview

Muscle SYNC is an AI-based sensorless muscle activation analysis system that uses computer vision to analyze muscle activity without requiring any physical sensors. By processing video input, the system detects and quantifies muscle activation patterns in real time.

Key Features

Sensorless Analysis: Eliminates the need for EMG sensors or wearable devices by leveraging computer vision and AI-based inference directly from camera input.

Real-time Processing: Performs muscle activation analysis in real time, enabling live feedback during exercise or rehabilitation sessions.

HW-SW Co-design: Optimized across both hardware and software layers for efficient deployment on edge platforms.
